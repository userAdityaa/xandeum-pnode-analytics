# About Xandeum Analytics

Xandeum Analytics is a comprehensive analytics platform designed to provide deep insights into the Xandeum network. The application offers real-time and historical data visualization, node analysis, network health monitoring, and resource tracking.

## Key Features

- **Node Analysis:** Explore detailed statistics and trends for pNodes and the overall network.
- **Network Health:** Monitor the health and synchronization status of the network in real time.
- **Snapshots:** Access and analyze network snapshots for historical data.
- **API Access:** Export data in CSV and JSON formats for further analysis.

Xandeum Analytics is built to empower users, developers, and network participants with actionable data and intuitive visualizations.
